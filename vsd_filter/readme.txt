■ ファイル構成
vsd_filter.auf … AviUtl 用プラグイン
sample.log … サンプルのログファイル
vsd_filter\ 以下 … ソースファイル

■ 使い方
1. vsd_filter.auf を AviUtl.exe と同じ場所にコピーしてください．
2. AviUtl を起動して，適当な動画を開きます．
3. メニューの ファイル→インポート→VSDメーター合成 を選び，ログファイ
   ルを選択してリードします．
4. メニューの フィルタ→VSDメーター合成 にチェックを入れます．
5. 動画とログの同期を取ります．
   動画の，あるフレームがログの何行目に対応するかを，何とかして (^^; 
   調べます．これを 2箇所 (走行開始前/後 等) について調べます．
   
   そうしたらメニューの 設定→VSDメーター合成の設定 を選び，v先頭 / v
   最後に，先ほど調べたフレーム番号の 先/後 を設定し，log先頭/log最後
   に，先ほど調べたログ番号の 先/後 を設定します．ログ番号は 0 から始
   まるので，1行目を指定したいときは 0 を設定してください．
   
※ aviutil の設定スライダは，max 999 までしか使えないので，ひとつの設
   定項目が 2つのスライダに分割されています．上のスライダは 数値 1 で 
   設定値 100 に相当します．すなわち，12345 を設定したいときは，上のス
   ライダに 123 を，下のスライダに 45 を設定してください．

6. 後は avi ファイルの書き出しを行ってください．

■ ログの形式
1行が 1サンプリングのテキストファイルで，各項目は TAB 1個で区切られて
います．各項目の順番は，左から

タコ [rpm]
スピード [km/h]
走行距離 [m]
加速度(前後)，6659.7/1G, 加速すると値が増える
加速度(左右), 6591.6/1G, 右旋回で値が増える

となっています．ただし，加速度はセンサーで読み取った値を直接ログに記録
する事を想定していたので，1G あたりの値が特殊です．
また，ログの先頭から 30サンプル分の値を 0G 時のセンサー値と仮定して使
用します (ので，静止状態からログをとる必要があります)．
